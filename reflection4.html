Reflection #4





QUIZ#1

I thought the quiz of Ms. Jen would be a quiz that had identification, multiple choices, etc. But our quiz is an online website name “User Inyerface” that we need to do the instructions and has a time limit. I thought that it will be easy for me, but I was wrong. It took me several minutes to finish because it was very very frustrating. It is an example of a (very) bad website interface. Upon finishing the quiz, I realized when I become a web developer (or someone who creates UI on websites), I need to be organized and to make sure that the users of the website will be comfortable and accessible for them.

Also, when giving instructions it needs to be specific, and not contradicting to it, because it may confuse the users. Honestly, I want to go to close the website because it is very frustrating but it is a quiz so I need to finish it. It was a fun experience though. I also get it why Ms. Jen is trying this website for us in order to know what are the Do’s and Dont’s when designing and making an user interface.
